Detailed Description

This class represents a Data Stream offering the related services A Data Stream is a destination for published events.

Publishers with appropriate permissions can write events directly to Data Streams. A Universal Messaging client session can optionally have a Data Stream, and receive events through it. A Data Stream can be a member of one or more Data Groups.
