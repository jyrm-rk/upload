Detailed Description

This class represents the logical connection between a com.pcbsys.nirvana.client.nSession and a realm server.

It encompasses all stores, handlers and state that has context between a fConnection and a server as opposed to a nSession.

void com.pcbsys.nirvana.base.clientimpl.singleconnection.ClientConnectionManagerImpl.releaseRequests ( )

This method will release all outstanding events waiting for response from the server, except for: nPopQueue, command=close.

It is detramental to those outstanding events state so use carefully.

Use nEventProcessorImpl#releaseRequests() to clear all outstanding events (for example if closing the connection manager).
