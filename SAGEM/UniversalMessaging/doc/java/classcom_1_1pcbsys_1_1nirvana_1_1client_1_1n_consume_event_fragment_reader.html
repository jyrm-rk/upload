Detailed Description

Reconstructs fragmented events back into one nConsumeEvent.

If the events have not been fragemented then they are passed directly to the nEventListener that was supplied in the constructor.

See Also
nConsumeEventFragmentWriter
Deprecated:
Message Fragment is not supported in UM, hence this class is deprecated as of version 10.7 and will be removed in the subsequent release

void com.pcbsys.nirvana.client.nConsumeEventFragmentReader.go ( nConsumeEvent  evt)

This method is called by the client for each event that the server sends.

It will check the events for fragemented events and will then reconstruct the original event which was fragemented by nConsumeEventFragementWriter.
