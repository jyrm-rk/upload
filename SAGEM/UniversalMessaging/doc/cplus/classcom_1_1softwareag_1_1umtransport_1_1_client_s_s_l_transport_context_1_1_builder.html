Detailed Description

This Builder object is used to build a ClientSSLTransportContext. The ClientSSLTransportContext is configured using the builder so that once build() is called, the ClientSocketTransportContext is only returned if it is valid. You can use the Builder to create simple readable code like this:

context = ClientSSLTransportContext::Builder()
.withHost("localhost")
.withPort(8080)
.build();

This Builder object is used to build a ClientSocketTransportContext. The ClientSocketTransportContext is configured using the builder so that once build() is called, the ClientSocketTransportContext is only returned if it is valid. You can use the Builder to create simple readable code like this:
