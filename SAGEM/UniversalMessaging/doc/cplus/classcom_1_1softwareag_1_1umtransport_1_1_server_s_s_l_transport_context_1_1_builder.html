Detailed Description

This Builder object is used to build a ServerSSLTransportContext. The ServerSSLTransportContext is configured using the builder so that once build() is called, the ServerSSLTransportContext is only returned if it is valid. You can use the Builder to create simple readable code like this:

context = ServerSSLTransportContext::Builder()
.withHost("localhost")
.withPort("9443")
.build();
