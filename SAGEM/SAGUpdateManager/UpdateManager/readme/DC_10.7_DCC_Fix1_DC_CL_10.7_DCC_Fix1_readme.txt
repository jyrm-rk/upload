1.0 Fix Name

webMethods Database Component Configurator 10.7 Fix 1.



2.0 Fix ID

DC_10.7_DCC_Fix1




3.0 Product(s)/Component(s) Affected

This fix affects the webMethods Database Scripts component of webMethods Database 
Configuration.



4.0 Requirements
None.



5.0 Platform/RDBMS Support

This fix is supported on all platforms supported by webMethods Product Suite 10.7


6.0 Cautions and Warnings

None.



7.0 Fix Contents

WDA-1742
Added support for "ReAuthenticateUser" option for using Kerberos authenticated database


8.0 Installation

Install the fix using the Software AG Update Manager. For instructions, see 
Using the Software AG Update Manager located either in the _documentation 
directory or on the documentation Web site at http://documentation.softwareag.com.



9.0 Uninstallation

Uninstall the fix using the Software AG Update Manager. For instructions, see 
Using the Software AG Update Manager.

NOTE: This uninstall procedure can only be used to uninstall the most recently 
installed fix. This action will revert your installation to the previously 
installed fix. You cannot apply this uninstall procedure to the previously 
installed fix.


 
9.1 Localization

This fix does not require an updated Language Pack. It might 
contain new messages and these messages will appear in English.
